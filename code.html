Here is some code that I wrote when I was trying to "brute force" through 100 combinations of numbers to find three numbers that equal 2020 and then multiply those numbers together to get a final answer in C++.

I am self taught in this language and I still have a long way to go but it is so far one of my favorite languages I have worked with. 

using namespace std; 
int solvePairs(int arr[], int n, int sum) {
for (int i = 0; i < n; i++) 
for (int j = 0; j < n; j++) 
for (int x = 0; x < n; x++) 
if (arr[i] + arr[j] + arr[x] == sum) 
return arr[i] * arr[j] * arr[x];
}
